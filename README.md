# MiOS
My project to try things and learn about low level programming.

### Compiling
Nasm, gcc and Makefile

### Features
  - Basic memory management functions
  - Get info of floppydrives
  - RTC time and date
  - PIT and PC speaker driver
  - Keyboard driver
